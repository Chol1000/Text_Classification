# SDG 3 Indicator Text Classification

Multi-label NLP system for predicting SDG 3 indicator relevance.

## Setup
1. Clone this repo
2. Upload Devex_train.csv and Devex_test_questions.csv to your Google Drive at:
   MyDrive/Text_Classification/data/
3. Open SDG3_Text_Classification.ipynb in Google Colab
4. Run all cells top to bottom

## Dependencies
pip install -r requirements.txt

## Output
submission.csv is saved to outputs/submission/
